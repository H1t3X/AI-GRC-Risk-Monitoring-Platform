# Threat Catalog

## Purpose

Defines threats monitored by the platform.

| Threat ID | Threat Name | Description |
|------------|-------------|-------------|
| T001 | DDoS Attack | Excessive traffic causing service disruption |
| T002 | Brute Force Attack | Multiple login attempts to gain access |
| T003 | Unauthorized Access | Access by unapproved users |
| T004 | Missing Monitoring | Lack of logging and visibility |
| T005 | Credential Theft | Stolen usernames and passwords |
| T006 | Data Exposure | Unauthorized disclosure of information |

---

## Threat Categories

### Availability Threats
- DDoS
- Service disruption

### Confidentiality Threats
- Unauthorized access
- Credential theft

### Integrity Threats
- Data manipulation
- Unauthorized changes
